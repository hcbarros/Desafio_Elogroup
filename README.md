
A pasta "css" possui imagens, uma fonte extra, arquivos auxiliares do Jquery e um arquivo index.css de define algumas características da página.

Na pasta "js" possui um arquivo index.js com as rotinas necessárias e dois arquivos Jquery auxiliares.

O arquivo index.html corresponde à página que será exibida com o formulário para envio dos dados.

O index.js que está na raiz do projeto cria o servidor local e a rota para envio dos dados.

O package.json possui as dependências necessárias para criação do servidor e rota.

Para executar siga os passos abaixo:

1)- Pela linha de comando, entre na raiz do projeto e digite "npm install" para instalar as dependências (node_modules)
2)- Em seguida, digite "node index.js" para criar o servidor e a rota.
3)- Ainda com o servidor em funcionamento, clique no arquivo index.html para exibir a página com o formulário e fazer todos os testes de validação e envio dos dados.
